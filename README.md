# 🚀 API Testing & Monitoring Tool

## 📌 Overview
This project is a web-based API Testing and Monitoring Tool that allows users to send API requests, analyze responses, and visualize performance. It also supports importing Postman collections for dynamic API execution.
## 🌐 Live Demo
👉 [https://api-testing-and-monitoring-9p8q.vercel.app/](https://api-testing-and-monitoring-9p8q.vercel.app/)
---
## ✨ Features
- 🔗 Send API requests (GET, POST)
- 📊 View response status, time, and data
- ⚡ Speed classification (Fast / Medium / Slow)
- 📈 Graph visualization of response time
- 📜 History tracking using LocalStorage
- 📂 Import Postman collections (JSON)
- 🎨 Clean UI with Dark Mode
---
## 🧩 How It Works
1. Enter API URL or import Postman collection
2. Select request method (GET/POST)
3. Click **Send**
4. View:
   - Status code
   - Response time
   - Speed category
   - Response data
---
## 🔗 Postman Integration
- Export collection from Postman (v2.1 JSON)
- Upload in the tool
- Click any API to auto-fill details
- Execute request instantly
---
## ⚙️ Tech Stack
- **Frontend:** HTML, CSS, JavaScript
- **Backend:** Serverless API (Node.js)
- **Charts:** Chart.js
- **Deployment:** Vercel
---
## 📊 Performance Measurement
The tool measures **frontend response time**, which represents real user-perceived latency, including backend and network delays.
